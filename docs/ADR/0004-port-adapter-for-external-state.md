# ADR-0004: 外部状態の取得を Port/Adapter で抽象化する

- ステータス: 採用
- 日付: 2026-06-30

## 背景
外部（クラウド）の状態取得に直接依存すると、テストやデモで実環境が必要になり、
実装の差し替えも難しくなる。

## 決定
外部状態の取得を Port（インタフェース）として定義し、サービスは Port にのみ依存する。
既定は手入力の結果を返す実装、特定のプロファイル時のみ実アダプタに切り替える。

## 検討した代替案
- サービスから直接 SDK を呼ぶ: テストが難しく、実装を差し替えられない。

## 結果
- 実環境なしでテスト・デモが回る。
- 実装を後から差し替えられる。
