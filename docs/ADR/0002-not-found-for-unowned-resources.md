# ADR-0002: 所有権のないリソースは404で応答する

- ステータス: 採用
- 日付: 2026-06-30

## 背景
他人のリソースへの要求に 403（権限なし）を返すと「存在はする」ことが伝わり、
識別子の総当りで存在の有無を推測されうる。

## 決定
所有権のないリソースへの閲覧・編集要求は、403 ではなく 404 で応答し、
「存在しない」と「権限がない」を区別せず存在自体を秘匿する。
一方、操作権限の不足（状態を変える操作など）は 403 で明示し、軸を分ける。

## 検討した代替案
- すべて 403 で統一する: 存在の有無が漏れる。
- すべて 404 にする: 操作権限の不足まで隠れ、利用者に不親切。

## 結果
- 識別子の総当りで存在を推測されにくい。
- 閲覧・編集経路は 404 をテストで保証する。
